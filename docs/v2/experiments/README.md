# V2 Experiments And Calibration Ledger

This directory is reserved for measured outcomes.

Use it for:

- fixture-trial results
- calibration summaries
- replay recommendation artifacts
- benchmark comparisons
- adoption evidence for model or policy changes

Do not use it for:

- stable subsystem theory
- generic roadmap language
- architecture decisions without measurement

Minimum expectations for an experiment record:

- date
- trial or config identity
- baseline reference
- artifact roots or report paths
- recommendation
- key caveats

If the question is "what happened when we tried this?", the answer belongs here.
